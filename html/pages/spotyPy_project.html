About the Project
Task

This project initially startet with the intention of learning python and my spotify passion.

Functionality

This project is based on Python. Its main function is to automatically edit playlists in Spotify. With a given seed from different databases the bot collects many new and popular songs. This has the huge advantage of getting the best collection of songs from the huge amount of different sources. Every Saturday at midnight, the bot then sorts the list of songs by release date, deletes duplicates, truncates the list to a total of 100 songs and automatically adds them to Spotify. This process can be done for any genre and is not limited by the number of databases. Future plans include integrating the popularity coefficient of users, increasing the number of databases and implementing a web interface.
