Selfhosted-Homeserver

About the Project
Task

The Task of the this project was to deploy my own homeserver for all my smarthome devices on a raspberry pi.

Functionality

This selfhosted homeserver offers the adventages of having a way more faster connection between all smart home devices. Furthermore the because of the local approch no data gets passed through data centers in the cloud, wich opens up a whole new way of privacy. In companion with the mycroft project i can also controle all my devices through voice commands. With this companiation I can fully replace the google services or Other cloud providers for smarthome.
