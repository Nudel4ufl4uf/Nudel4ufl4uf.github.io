HomeAssistant

Self-Hosted Homeserver

About the Project
Task

The idea of this project was to deploy my own home server for all my smart home devices locally on a Raspberry Pi.

Functionality

This self-hosted home server provides the benefits of a much faster connection among all smart home devices. Furthermore, because of the local approach, no data gets passed through data centers in the cloud, which opens up a whole new way of privacy. In companion with the Mycroft project, I can also control all my devices through voice commands. With this combination, I can fully replace the Google Services or other cloud providers for smart home. The downside of this approach is the fact that it is impossible to reach the system from outside the network without any special customization. This limits the system to specific automations and other functionality that require data from outside the network. Nevertheless, the improved privacy and security is more important to my because all the core requirements to the system are met.
