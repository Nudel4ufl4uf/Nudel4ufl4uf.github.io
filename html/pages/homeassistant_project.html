HomeAssistant

Self-Hosted Homeserver

About the Project
Task

The idea of this project was to deploy my own home server for all my smart home devices on a Raspberry Pi.

Functionality

This self-hosted home server provides the benefits of a much faster connection among all smart home devices. Furthermore, because of the local approach, no data gets passed through data centers in the cloud, which opens up a whole new way of privacy. In companion with the Mycroft project, I can also control all my devices through voice commands. With this combination, I can fully replace the Google services or Other cloud providers for smart home.
