About the Project
Task

This project was my bachelor thesis from 2023. The main goal was to implement a Software Add-on to provide a documentation toolbox in the AR space of the Microsoft HoloLens. Furthermore, the software add-on was evaluated with the Port of Kiel to analyse the prototype in its functionality and possible use cases.

Functionality

The software Add-on has five major functionality to provide a seamless documentation process:

(1) Draw

(2) Erase

(3) Text

(4) Save/Load

(5) Settings

With these core functions the user is able to document any give business process within or outside as company.
